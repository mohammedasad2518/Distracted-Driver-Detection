# Distracted Driver Detection 🚗📱

A Deep Learning-based Distracted Driver Detection system that identifies unsafe driving behaviors using Computer Vision and CNN-based image classification techniques. The project is built using TensorFlow and Keras with a ResNet-50 inspired architecture to improve road safety through intelligent driver monitoring.

---

## 📌 Overview

Distracted driving is one of the leading causes of road accidents worldwide. This project focuses on detecting different driver activities such as texting, talking on the phone, eating, reaching behind, and safe driving using image classification techniques.

The model analyzes driver images and predicts the driver’s behavior category, helping in the development of smart transportation and accident prevention systems.

## 📊 Dataset

The dataset contains thousands of driver images categorized into multiple distraction classes, including:

- Safe Driving
- Texting
- Talking on Phone
- Eating
- Reaching Behind
- Operating Radio
- Hair & Makeup

Images were resized, normalized, and converted into arrays for model training.

---

## 🧠 Model Architecture

The project uses a ResNet-50 inspired CNN architecture consisting of:

- Convolutional Layers
- Batch Normalization
- ReLU Activation
- Max Pooling
- Residual Connections
- Fully Connected Layers
- Softmax Classification Layer

The model was trained using the Adam optimizer for multi-class image classification.
